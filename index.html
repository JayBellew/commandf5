The Command F5 Project

Encouraging Accessible & Inclusive Design and Development

Let's make our Websites and Apps Accessible to Everyone!

The Command F5 Project is a resource dedicated to encouraging Designers and Developers to create software that is accessible to all users.

Step 1: Listen

Spend some time using your site or app by listening through a screen reader and keyboard only.  You'll have an immediate appreciation for the challenges many of your users are having with your site.

Step 2: Learn

Learn as much as you can about designing and developing for folks with disabilities.  It's easier than you think.

How To Listen to Your Interface

For Mac Users

Every Mac comes standard with assistive technologies that can help you use macOS.

The hot keys to open the Mac VoiceOver are Command Key + F5

macOS also features an Accessibility Options window that lets you quickly turn on or off common accessibility features like Zoom, VoiceOver, Sticky Keys, and more. You can bring up this window anytime with a simple shortcut.
  • On the built-in keyboard of a Mac notebook computer, press Function-Option-Command-F5.
  • On a Mac notebook with Touch ID (power button)*, triple-press Touch ID.
  • On an external keyboard, press Option-Command-F5.

* Info from the Mac Support Site

For Windows PC Users

The Windows operating system includes several accessibility features that make using Windows easier, for people with age-related or other disabilities. By the time folks cross the fifties, most will have vision, hearing, or dexterity impairments.

The hot keys vary based on your PC and it's operating system but try this: Hit the Windows Logo Key + U. This should open your Ease of Access Options.

Otherwise try Windows Logo Key + Enter to start the Narrator

* Info from the Windows Support Site
